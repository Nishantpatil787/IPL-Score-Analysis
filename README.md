# IPL Score Analysis

This project analyzes IPL match data using Python.

## Objectives
- Analyze team performance
- Highest scoring matches
- Best batsmen
- Best bowlers
- Win percentage by teams

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
IPL dataset containing match and ball-by-ball data.

## Sample Analysis
- Total runs by season
- Top run scorers
- Toss impact on match result
